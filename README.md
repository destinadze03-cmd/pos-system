**POS System** is a web-based point-of-sale application designed for managing a retail shop.  
It allows managing **products, categories, sales,reports and receipts** through a clean and responsive web interface built with Laravel.

---

## Features

- Manage products and categories
- Record sales and generate receipts
- User authentication (login/register)
- Dashboard with statistics
- Responsive design for desktop and mobile
- Search, filter, and manage inventory efficiently
